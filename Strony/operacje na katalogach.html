Moduł os, od operating system (system operacyjny) pozwala wykonywać operację, za które odpowiedzialny jest system operacyjny. Czyli:

  • operacje na systemie plików (wyświetlanie zawartości folderów, usuwanie plików, zmiany uprawnień, tworzenie katalogów itp)
import os
files = os.listdir('.')
current_dir = os.getcwd()
print(f"Pliki z katalogu {current_dir}: {files}")
os.mkdir("nowy_katalog")
os.rmdir("nowy katalog")
  • operacje na procesach - kończenie procesów, startowanie nowych
  • różne różności jak np odczytanie ilości watków procesora: 
    import os
cpu_count = os.cpu_count()
print(f"Masz {cpu_count} watkow")
